## Getting Started

### Step 1: Start the Metro Server

To start Metro, run the following command from the root of your React Native project:

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

# using npm

npm run android

# OR using Yarn

yarn android

# using npm

npm run ios

# OR using Yarn

yarn ios
